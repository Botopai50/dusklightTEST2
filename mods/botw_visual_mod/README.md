# Breath of the Wild Visual Pass

This bundled mod complements the renderer-integrated cel shading with effects that belong in a
screen-space pass: color grading, shadow lift, atmospheric depth fog, selective bloom, lightweight
contact AO and a subtle water response. Every expensive component has an in-game intensity control.

The material shader remains responsible for hard light bands, cool indirect light, warm direct
light, stylized specular highlights and restrained back-light rim lighting.
